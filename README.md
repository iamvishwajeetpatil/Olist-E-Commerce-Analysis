🛒 Olist E-Commerce Analysis
📌 Project Overview

This project analyzes the Brazilian Olist E-Commerce dataset to uncover customer behavior, delivery performance, payment trends, and review sentiment.

The analysis focuses on:

Exploratory Data Analysis (EDA)
Customer purchasing behavior
Delivery performance insights
Statistical hypothesis testing
Business recommendations for leadership
📂 Dataset Used

The dataset contains multiple relational tables including:

Customers
Orders
Products
Order Items
Payments
Reviews

Data Source: Olist Brazilian E-Commerce Public Dataset

Analyzed:

Non-Intuitive Finding:

Higher freight charges did not always lead to poor customer reviews.

🧪 Hypothesis Testing
Hypothesis 1
Question:

Do delayed deliveries reduce customer review scores?

Result:

✅ Accepted

Customers receiving delayed deliveries gave significantly lower ratings.

Hypothesis 2
Question:

Do expensive freight charges reduce customer satisfaction?

Result:

❌ Weak relationship observed

Freight value alone was not a major driver of dissatisfaction.

💡 Business Recommendations
1️⃣ Improve Delivery Operations

Focus on reducing delayed deliveries to improve customer satisfaction.

2️⃣ Optimize Logistics Costs

Reduce freight charges in high-cost regions using warehouse optimization.

3️⃣ Improve Customer Communication

Provide proactive delivery updates to reduce negative reviews.
